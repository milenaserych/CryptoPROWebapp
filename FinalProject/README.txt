CryptoPRO


Installation: 

Run the following command to install all of the packages used in the project
pip install -r requirements.txt


Usage:

Run the Django development server to open the app on http://localhost:8000
python manage.py runserver



Description:

CryptoPRO is a web application introducing the basics of cryptocurrencies to beginners.
Users interested in the crypto market but feel intimidated by the complexity of the terms and definitions are the application's target audience.
People using CryptoPRO will learn the required foundations in an easy and engaging way.

The application is currently at the final prototype stage but has not been deployed yet.
